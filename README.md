## Learning Outcomes

This course provides the foundational knowledge of computer programming. Students will learn how to:

- Analyze simple problems and solve them with procedural algorithms
- Use C++ as an imperative programming language
- Solve simple data analysis and numerical calculation problems

### After completing this course, you will be able to:

- Grasp **Computational Thinking** principles effectively
- Understand **imperative programming** concepts
- Utilize **C++** (non-object-oriented concepts)
- Design, implement, and validate small sequential programs
- Write small-scale C++ programs with correct and comprehensible code
- Operate an **IDE** (e.g., VS Code) comfortably

### Topics Covered

- Pseudocode and flowcharts
- Syntax and semantics of a programming language
- Basic elements of formal grammars
- Introduction to visual programming (Blockly)
